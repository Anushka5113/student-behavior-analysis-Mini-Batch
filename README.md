This project uses the MiniBatch K-Means Clustering algorithm to efficiently cluster students based on their academic and behavioral attributes. MiniBatch K-Means is particularly suitable for large datasets because it processes small random batches of data, reducing training time while maintaining good clustering quality.
Project Objectives:-
Analyze student academic performance and behavioral patterns.
Group students with similar characteristics using clustering.
Identify learning behavior trends.
Support educational decision-making through data analysis.
Demonstrate scalable unsupervised machine learning techniques.
